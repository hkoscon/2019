The 8 essential steps to secure your MySQL

event 2019-06-15 (Day 2) ~ 10:00 - 10:30
explore Conference Hall 5
  • comment English
  • network_check Beginner

There are 8 categories in securing Database - starting with general risks
and concluding with Enterprise features.

  1. Password
  2. Anonymous account and `test` database
  3. Account and Privileges
  4. Secure Connection and anti-Brute force attach
  5. Monitoring system variables
  6. Filesystem
  7. Encryption
  8. Enterprise level security configuration

Ryusuke Kajiyama

Oracle / Japan

Ryusuke Kajiyama ( 梶山隆輔 ) is Open Source Specialist with 10+ years of experience of systems design using MySQL and open source solutions. He originally joined MySQL AB, and is currently managing MySQL Sales Consultant team of Asia Pacific region at Oracle.
